# Zeno - Unified Digital Ecosystem

A cohesive suite of integrated applications designed for productivity and privacy.

## Live Demo
Coming soon at: [username.github.io/zeno](https://username.github.io/zeno)

## Applications
- **Zeno Home** - Unified dashboard with customizable widgets
- **Zeno Mail** - Privacy-focused email client
- **Zeno Calendar** - Smart scheduling with integrations
- **Zeno Cloud** - Secure file storage and sharing
- **Zeno Connect** - Team messaging and collaboration
- **Zeno Maps** - Clean, distraction-free navigation

## Development
This project is built with vanilla HTML, CSS, and JavaScript, requiring no installation or build tools.

### Quick Start
1. Open in GitHub Codespaces

3. Start developing!
